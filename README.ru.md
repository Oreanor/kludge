# AIR - Расширение для VS Code с поддержкой ИИ

Расширение для VS Code, которое добавляет интеллектуальный чат в боковую панель Explorer для удобного взаимодействия с ИИ прямо в редакторе.

## Возможности

- 💬 Интерактивная панель чата в боковой панели VS Code
- 🎨 Современный UI на базе React
- 🔌 Легко расширяется под разные ИИ провайдеры
- 🔒 Безопасная обработка контента с CSP
- ⚡ Обмен сообщениями в реальном времени

## Установка

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/yourusername/air.git
   cd air
   ```

2. Установи зависимости:
   ```bash
   npm install
   ```

3. Собери расширение:
   ```bash
   npm run build
   ```

4. Открой VS Code и нажми `F5` для отладки, или упакуй для релиза.

## Использование

1. Открой VS Code с загруженным расширением
2. Найди панель **AIR Chat** в боковой панели Explorer
3. Введи сообщение и нажми Enter
4. Ответ появится в истории чата

## Разработка

### Основные команды

- `npm run build` - собрать расширение и webview
- `npm run watch` - следить за изменениями и пересобирать
- `npm run compile` - собрать только расширение
- `npm run build:webview` - собрать только webview
- `npm run lint` - проверить код
- `npm run test` - запустить тесты

### Структура проекта

```
src/              - Код расширения (TypeScript)
webview/          - UI на React
dist/             - Собранные файлы (генерируется автоматически)
```

### Как делать изменения

- **Логика расширения**: редактируй файлы в `src/`
- **UI чата**: редактируй файлы в `webview/src/`
- Запусти `npm run watch` для автоматической пересборки
- Нажми `Ctrl+R` в окне отладки для перезагрузки

### Добавление нового ИИ провайдера

Отредактируй файл `src/chatProvider.ts`, метод `_handleMessage`, чтобы добавить свой сервис ИИ.

## Отладка

1. Нажми `F5` для открытия нового окна VS Code с расширением
2. Открой панель чата для тестирования
3. Используй `Ctrl+Shift+I` для DevTools webview
4. Используй консоль отладки VS Code для логов расширения
